## Descripcion

Este repositorio contiene todos los entregables del Grupo 4 para la Tarea 3 del curso de Ingenieria de Software I. El grupo tiene asignada la documentacion de tres bloques: Plan de Pruebas, Documentacion DevOps y Documentacion de Usuario, todos ilustrados con ejemplos del Sistema de Gestion Academica (SGA).


## Contenido del repositorio
| Archivo | Descripcion |
| `documento_principal.pdf` | Documento principal de 7 paginas con definicion, proposito y ejemplos de cada tipo de documentacion asignada |
| `anexos/runbook_operacional.md` | Runbook con guia de incidentes, metricas y procedimientos operacionales del SGA |
| `presentacion.pptx` | Diapositivas para la defensa oral |


| Criterio | Como se aborda |
| Exactitud y profundidad | Cada seccion define proposito, secciones clave y cuando usar cada documento |
| Calidad de plantillas | El runbook es una plantilla editable y aplicable a cualquier proyecto similar |
| Organizacion y redaccion | Lenguaje tecnico profesional con referencias reales en formato APA |
| Presentacion oral | Diapositivas en archivo adjunto |
